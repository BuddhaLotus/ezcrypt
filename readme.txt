EZCypt----易密，轻松保密你生活！

软件名称：
	英文：EZCrypt
	中文：易密

软件类型：
	加密软件

功能简介：
	通过创建加密数据容器文件并可动态加载为分区，提供一个简单易用的加密
手段，用户需要加密的数据直接拷贝、移动到此分区中即可在复制、移动过程中
透明实现高强度加密（支持多种加密方式，并将不断扩充），加密速度快、对用
户透明（用户无感知）、简单易用。

特性列表：
	绿色，免安装；
	体积小巧，携带方便（只需要3个文件，总共不到400K，够迷你了吧？！）；
	透明加密，使用简便（Encrypt On The Fly）；
	加密数据容器加载为分区，直观易用；
	加/解密速度快，对复制、移动数据没有加/解密瓶颈（14.8MB/S AES128 
		在奔四1.4G主机上测试）；
	加密强度高，支持包括AES、TwoFish等高强度加密算法，安全性极高；
	特殊设计的加密容器文件格式，用户密码等采用高强度Hash算法加密后存储，
		几乎不可解密，所有关键数据均另外采用高强度加密算法加密后存储，更
		加安全可靠；
	特殊设计的加密容器文件格式，不存在任何明/密文标记，文件格式不可分析
		避免了加密算法攻击；
	特殊设计的加密容器文件格式，其中文件头部保存关键信息，除此之外没有其
		它地方保存加密信息，更加安全可靠；
	界面简洁易用；
	支持快捷键操作方式；
	支持隐秘运行方式，更加保密；
	支持配置文件，用户可以实现功能配置；
	优化、高效的驱动设计，可实现动态加载/卸载，减少资源占用；
	可以充分利用NTFS分区的文件系统特性，创建可扩展文件（即在容器刚刚创建
		的时候，文件很小，随着其内容的不断增多，实际占用的空间才越来越大），
		大幅降低空间浪费；
